# TypeChain

TypeChain is a small blockchain in TypeScript.

## To Run the Script
`$ yarn start`

## To Create a New Block
```typescript
createNewBlock("N-th block");
```
Write the new block's name between `createNewBlock`'s parentheses.

## Output
`console.log(blockchain);`
